# TinyTorch

An Auto-Diff Optimization Framework for Teaching and Understanding Pytorch

Cpp part Source from [TinyTorch](https://github.com/darglein/TinyTorch)

## Building TinyTorch

```shell
cd TinyTorch
mkdir build && cd build
cmake ..
make
```
